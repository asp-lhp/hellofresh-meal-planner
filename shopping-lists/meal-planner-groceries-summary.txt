OPTION 1: Quick Setup with Anthropic API Key
---------------------------------------------
1. Get API key from: https://console.anthropic.com/settings/keys
2. Configure OpenClaw:

   openclaw onboard --install-daemon

   (Follow prompts to add your Anthropic API key)

3. Test with your shopping list:

   openclaw agent --local --session-id groceries \
     --message "Analyze this shopping list and suggest optimizations" \
     < meal-planner-groceries.json


OPTION 2: Use Existing OpenClaw Skills
---------------------------------------------
Install community skills that work with grocery lists:

# Grocery list manager
openclaw skill install anylist

# AI shopping concierge
openclaw skill install clawringhouse

# Use with your export
openclaw agent --skill anylist \
  --message "Add these to my grocery list" \
  < meal-planner-groceries.json


OPTION 3: Export Only (No OpenClaw Processing)
---------------------------------------------
If you just want the JSON export for other tools:
